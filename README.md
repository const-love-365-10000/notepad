## 编译

Electron-package使用方法：

`electron-packager <sourcedir> <appname> --platform=<platform> --arch=<arch> [optional flags...]`

例如：

`electron-packager . notepad  --win --out ./dist --arch=x64 --app-version=0.0.1 --electron-version=2.0.0`
